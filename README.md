# DevOps Noir Portfolio

A high-performance, minimalist professional portfolio designed for Cloud and DevOps engineers. Built with **Next.js 15 (Turbopack)**, **Tailwind CSS**, and **Framer Motion**.

## üöÄ Features

-   **Cinematic Terminal Hero**: Dynamic, looping terminal session showcasing real DevOps workflows.
-   **Infrastructure Pipeline Visualization**: Smooth scroll-triggered reveals of technical expertise.
-   **Interactive Skill Cloud**: Infinite-scrolling marquee of technical competencies.
-   **Multi-tenant Project Sub-pages**: Detailed dynamic routes for every project in the portfolio.
-   **DevOps Noir Aesthetic**: Specialized obsidian and emerald green color palette with glassmorphism effects.
-   **Smooth Navigation**: Integrated Lenis scroll for a premium, high-end user experience.
-   **Mobile First**: Fully responsive design with an interactive mobile navigation menu.

## üì¶ Getting Started

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/talatops/DevOps-Portfolio.git
    cd DevOps-Portfolio
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    ```

3.  **Run the development server**:
    ```bash
    npm run dev
    ```

4.  **Build for production**:
    ```bash
    npm run build
    ```
